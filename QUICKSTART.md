# 快速开始指南

## 5分钟快速部署

### 步骤 1: 安装依赖

```bash
# 安装所有依赖（根目录、前端、后端）
npm run install:all
```

### 步骤 2: 构建项目

```bash
# 构建前端和后端
npm run build
```

### 步骤 3: 启动服务

```bash
# 开发模式（自动重启）
npm run dev

# 或生产模式
npm start
```

### 步骤 4: 访问系统

打开浏览器访问：`http://localhost:9667`

## 使用说明

### 1. 配置保存

在前端界面中：
1. 填写用户名和密码
2. 填写工作日期列表（每行一个日期，格式：YYYY-MM-DD）
3. 填写项目ID、工时等信息
4. 点击"保存配置"按钮

配置会自动保存到本地数据库（`backend/data/db.json`）

### 2. 查看任务状态

- 系统会自动每分钟执行一次填写工时任务
- 在"任务状态"区域可以查看：
  - 最后执行时间
  - 最后成功时间
  - 是否正在运行
  - 上次执行结果

### 3. 手动操作

- **手动填写工时**：点击"手动填写工时"按钮，立即执行一次填写
- **同步工作信息**：点击"同步工作信息"按钮，获取工作项目信息
- **刷新状态**：点击"刷新状态"按钮，更新任务状态信息

## 常见问题

### Q: 定时任务不执行？

A: 检查以下几点：
1. 配置是否完整（用户名、密码、工作日期列表、项目ID等）
2. 今天是否已经成功执行过（成功执行后当天不再执行）
3. 查看任务状态中的错误信息

### Q: 如何修改端口？

A: 编辑 `backend/src/config/config.ts`，修改 `port` 值

### Q: 如何查看日志？

A: 日志文件在 `backend/log/` 目录，按日期命名

### Q: 数据存储在哪里？

A: 所有数据存储在 `backend/data/db.json`
